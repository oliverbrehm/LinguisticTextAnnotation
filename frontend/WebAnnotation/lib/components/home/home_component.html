Prosodiya Online Text Analyse


Aufbereitung von Texten zur LRS Therapie

Die Online Text Analyse wurde im Rahmen einer Masterarbeit an der Universität Tübingen entwickelt. Die App gibt die Möglichkeit beliebige Texte zu analysieren und aufzubereiten.
Die ursprüngliche Zielsetzung war dabei Silben und Betonungsmuster in Wörtern farblich hervorzuheben. So bearbeitete Texte werden von Lerntherapeuten für die Therapie von Lese-Rechtschreibschwäche (LRS) eingesetzt. Durch die Trennung und Hervorhebung einzelner Silben kann der Sprachrhythmus im Text besser wahrgenommen werden, somit wird das Textverständnis verbessert.
Die App soll das Erstellen dieser Lerntexte vereinfachen. Im Vergleich einer manuellen Generierung der Texte, zum Beispiel mit einem Textverarbeitungsprogramm, geschieht das meiste automatisch. Bei Unklarheiten (z.B. unbekannte Wörter) wird der Nutzer durch eine übersichtliche Benutzerschnittstelle geleitet, um diese zu beseitigen. Durch die Verwendung von Nutzerkonten können Texte auch gespeichert werden, außerdem kann die globale Wort Datenbank erweitert werden.

Textanalyse

Zur Analyse kann einfach ein eigener Text eingegeben oder kopiert werden. Nachdem der Text analysiert wurde können unbekannte Wörter geklärt und die Anzeigeeinstellungen angepasst werden. Es stehen außerdem Funktionen zum Drucken oder Kopieren (Weiterverwendung in Textverarbeitungsprogramm) zur Verfügung.

Nutzerkonto

Zum effektiven Arbeiten mit der App wird ein Nutzerkonto benötigt. Dadurch lassen sich eigene Einträge für unbekannte Wörter erstellen sodass diese erkannt werden, wenn sie erneut vorkommen. Außerdem können verwendete Texte gespeichert und wiederverwendet werden.

Betonungsdatenbank

Das Herzstück der Textanalyse ist eine Datenbank in der Betonungsmuster und andere Worteigenschaften gespeichert sind. Wegen des immensen Umfangs eines Sprachwortschatzes sind nicht alle Wörter und Wortbeugungen der deutschen Sprache enthalten.
Die Datenbank kann jedoch von allen Nutzern erweitert werden. Wird ein unbekanntes Wort bei der Textanalyse hinzugefügt steht es nur dem Benutzer zur Verfügung, der den Eintrag erstellt hat. Um es allen Nutzern bereitzustellen, muss es von mindestens zwei weiteren Nutzern bestätigt werden. Auf der Seite Verifizierung kann jeder Nutzer Wörter bestätigen oder Gegenvorschläge einbringen. So kann die Datenbank wachsen und mehr Wörter werden automatisch erkannt.
